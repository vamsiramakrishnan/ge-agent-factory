# Critique Rubric

Score each category 0-3.

## Business Specificity

- 0: generic assistant
- 1: department named but workflow vague
- 2: persona, use case, and workflow are clear
- 3: mirrors source slide language and buyer-relevant KPI

## System / Mock Credibility

- 0: no systems or fake placeholders
- 1: systems listed but tools/data generic
- 2: tools reflect real system responsibilities
- 3: deterministic fixtures, evidence, errors, and audit trail exist

## ADK Correctness

- 0: not runnable
- 1: imports or project shape questionable
- 2: valid local ADK structure with tools
- 3: local run, eval, and extension points are clean

## Validation Strength

- 0: no checks
- 1: syntax only
- 2: smoke tests and fixture validation
- 3: evalset plus run log evidence

## Registration Readiness

- 0: confuses local tools with Gemini Enterprise
- 1: mentions deployment vaguely
- 2: separates local, Agent Registry, deployment, Enterprise
- 3: has publish plan with prerequisites and approval gates
