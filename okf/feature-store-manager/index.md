# Feature Store Manager

> IT6-04 • Data & AI Platform

## Overview

- **Persona:** Data Platform Lead
- **Department:** it
- **Objective:** Gemini recommends reusable features when data scientists describe their model use case. LLM detects redundant feature definitions across teams and recommends consolidation. so the Data Platform Lead can move the Feature reuse rate KPI.

## KPI summary

- **Feature reuse rate**: 15% reused across models → 60% with recommendations
- **Redundant features**: Unknown count → Identified and consolidated
- **Feature discovery time**: Days of asking around → Minutes via search
