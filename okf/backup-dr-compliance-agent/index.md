# Backup & DR Compliance Agent

> A-4008 • Infra & Cloud Ops

## Overview

- **Persona:** Cloud Architect / SRE Manager
- **Department:** it
- **Objective:** Gemini identifies backup coverage gaps immediately — catching systems dropped from backup jobs during migrations. Continuous RPO/RTO compliance scoring replaces quarterly audits with daily validated readiness. so the Cloud Architect / SRE Manager can move the Backup coverage gaps KPI.

## KPI summary

- **Backup coverage gaps**: Found in DR drills → Daily detection
- **RPO/RTO compliance**: Quarterly audit → Continuous monitoring
- **DR readiness score**: Unknown → Daily scored by tier
