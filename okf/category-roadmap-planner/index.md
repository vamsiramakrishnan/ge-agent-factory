# Category Roadmap Planner

> A-1210 • Strategic Sourcing

## Overview

- **Persona:** Category Director
- **Department:** procurement
- **Objective:** Gemini reasons about sequencing: 'Consolidate the supply base before renegotiating — the volume leverage from 5 to 2 suppliers adds 12% negotiation power.' LLM identifies dependencies and risks in narrative form that a Category Director can present to leadership — not just a Gantt chart. so the Category Director can move the Roadmap creation time KPI.

## KPI summary

- **Roadmap creation time**: 3-4 weeks → 3 days
- **Initiative dependencies mapped**: Informal → Systematically tracked
- **Savings pipeline accuracy**: 50-60% realized → 80%+ realized
