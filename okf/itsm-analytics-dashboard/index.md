# ITSM Analytics Dashboard

> IT5-08 • IT Service Management

## Overview

- **Persona:** IT Service Desk Manager
- **Department:** it
- **Objective:** Gemini generates the weekly report automatically with trend commentary and anomaly explanations. LLM separates signal from noise — 'volume up 15% from MFA rollout, underlying demand flat.' so the IT Service Desk Manager can move the Report creation time KPI.

## KPI summary

- **Report creation time**: 4-6 hours manual → Automated weekly
- **Anomaly explanation**: Requires investigation → Auto-contextualized
- **Metric coverage**: 5 core KPIs → 25+ metrics with trends
