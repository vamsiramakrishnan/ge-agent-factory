# Inventory Valuation Agent

> A-2706 • Revenue & Cost Accounting

## Overview

- **Persona:** Cost Accountant
- **Department:** finance
- **Objective:** Item-level impairment assessment replaces blanket reserves with documented rationale per material. Gemini cross-references production plans and product discontinuation to identify true obsolescence. so the Cost Accountant can move the SLOB identification time KPI.

## KPI summary

- **SLOB identification time**: 1-2 weeks → < 1 day
- **Write-down accuracy**: Conservative blanket reserves → Item-level assessment
- **Inventory carrying cost**: 12-15% of revenue → 8-10% optimized
