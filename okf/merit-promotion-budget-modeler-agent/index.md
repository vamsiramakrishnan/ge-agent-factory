## Overview

- **Persona:** Comp Manager
- **Department:** hr
- **Objective:** Multi-scenario merit and promotion budget modeling with sensitivity analysis. Real-time cost projections by org unit, job family, and geography. so the Comp Manager can move the Scenario modeling KPI.
