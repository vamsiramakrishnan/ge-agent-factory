# Should-Cost Modeler

> A-1203 • Strategic Sourcing

## Overview

- **Persona:** Category Manager
- **Department:** procurement
- **Objective:** Gemini interprets engineering specs like 'investment casting with post-machining to 0.05mm tolerance' and maps them to specific cost drivers. LLM reasons about supplier premium justification — is it a capability premium or an inefficient cost structure? so the Category Manager can move the Model build time KPI.

## KPI summary

- **Model build time**: 2-3 weeks → 4 hours
- **Cost driver coverage**: Top 3 drivers → 12+ drivers modeled
- **Negotiation savings lift**: Baseline → +4-7% incremental
