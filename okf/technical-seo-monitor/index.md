# Technical SEO Monitor

> A-3206 • Digital Marketing & SEO/SEM

## Overview

- **Persona:** SEO/SEM Specialist
- **Department:** marketing
- **Objective:** Daily automated monitoring with instant Slack alerts for critical SEO issues. Gemini diagnoses complex issues — explaining why thin content triggers soft penalties, not just flagging the symptom. so the SEO/SEM Specialist can move the Issue detection speed KPI.

## KPI summary

- **Issue detection speed**: Weekly manual → Real-time
- **Crawl error resolution**: 5-7 days → Same day
- **Indexation rate**: 78% → 95%+
