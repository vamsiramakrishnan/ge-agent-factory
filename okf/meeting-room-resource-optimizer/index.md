# Meeting Room & Resource Optimizer

> A-4606 • End User Computing

## Overview

- **Persona:** End User Support Lead
- **Department:** it
- **Objective:** Gemini analyzes actual booking patterns, attendance, and no-show rates to identify space misallocation. LLM recommends specific reconfigurations backed by utilization data and cost-per-seat analysis. so the End User Support Lead can move the Room utilization rate KPI.

## KPI summary

- **Room utilization rate**: 42% → 78%
- **No-show rate**: 35% → 12% (with nudges)
- **Room booking complaints**: 40/month → 8/month
