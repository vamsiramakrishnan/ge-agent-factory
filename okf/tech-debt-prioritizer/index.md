# Tech Debt Prioritizer

> A-3903 • Software Engineering & DevOps

## Overview

- **Persona:** VP Engineering / DevOps Lead
- **Department:** it
- **Objective:** Gemini correlates code quality with incidents and change velocity to produce a data-driven priority ranking. Refactoring ROI estimation gives engineering leadership a business case for tech debt investment. so the VP Engineering / DevOps Lead can move the Debt prioritization method KPI.

## KPI summary

- **Debt prioritization method**: Gut feel → Data-driven scoring
- **Incident reduction from refactoring**: Unknown → 40% fewer repeat incidents
- **Sprint debt allocation**: 0-5% ad hoc → 20% systematic
