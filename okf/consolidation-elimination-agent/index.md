# Consolidation & Elimination Agent

> A-2806 • Finance Analytics & Reporting

## Overview

- **Persona:** Financial Reporting Manager
- **Department:** finance
- **Objective:** Automated entity data collection with currency translation reduces cycle time dramatically. 97% IC auto-matching with BlackLine integration minimizes manual investigation. so the Financial Reporting Manager can move the Consolidation cycle KPI.

## KPI summary

- **Consolidation cycle**: 3-5 days → < 1 day
- **IC elimination rate**: 80% auto-matched → 97% auto-matched
- **Conforming adjustments**: Manual calculation → AI-computed with documentation
