# Identity & Access Anomaly Detector

> IT4-05 • Cybersecurity & Threat Management

## Overview

- **Persona:** CISO / Security Analyst
- **Department:** it
- **Objective:** Gemini correlates identity anomalies with HR events in real time, catching orphaned accounts within hours. LLM contextualizes anomalies — distinguishing VPN routing artifacts from genuine credential compromise. so the CISO / Security Analyst can move the Anomaly detection to action KPI.

## KPI summary

- **Anomaly detection to action**: Days (if detected) → <1 hour
- **Orphaned account detection**: Quarterly audit → Real-time HR correlation
- **Insider threat coverage**: Reactive investigation → Proactive UBA monitoring
