# HiPo Identification & Nomination Agent

> A-310 • Performance Mgmt

## Overview

- **Persona:** CHRO
- **Department:** hr
- **Objective:** Multi-signal HiPo scoring using performance data, 360 feedback, and project impact. Bias-adjusted nominations with transparency into weighting criteria. so the CHRO can move the Identification method KPI.

## KPI summary

- **Identification method**: Manager nomination → Multi-signal scoring
- **Bias adjustment**: None → Automated
- **Criteria transparency**: Opaque → Explainable
