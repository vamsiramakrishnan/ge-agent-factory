# Marketing Dashboard Generator

> A-3402 • Marketing Analytics & Attribution

## Overview

- **Persona:** Marketing Analyst
- **Department:** marketing
- **Objective:** Gemini answers ad-hoc marketing questions in natural language with data, narrative, and charts. Daily automated dashboard refresh across all marketing systems with anomaly detection. so the Marketing Analyst can move the Dashboard refresh KPI.

## KPI summary

- **Dashboard refresh**: Manual weekly → Daily automated
- **Ad-hoc query time**: Hours of analysis → Natural language
- **Data source coverage**: 3-4 manual → All systems unified
