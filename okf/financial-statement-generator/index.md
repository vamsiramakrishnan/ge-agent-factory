# Financial Statement Generator

> A-2801 • Finance Analytics & Reporting

## Overview

- **Persona:** Financial Reporting Manager
- **Department:** finance
- **Objective:** Automated TB-to-statement mapping with validated balance sheet equation on close day. Gemini generates management discussion narratives with variance explanations tailored to materiality. so the Financial Reporting Manager can move the Statement preparation KPI.

## KPI summary

- **Statement preparation**: 2-3 days post-close → Same-day automated
- **MD&A drafting time**: 1-2 days → Auto-generated with review
- **Rounding errors**: Manual reconciliation → Zero — automated validation
