# Total Cost of Ownership Modeler

> A-1905 • Spend Analytics

## Overview

- **Persona:** Category Manager
- **Department:** procurement
- **Objective:** Multi-factor TCO spans 25+ cost dimensions: acquisition, logistics, quality, carrying, disposal, and risk premium. LLM quantifies hidden costs invisible in structured systems — 'Supplier B requires a dedicated engineering liaison (0.25 FTE at $150K).' so the Category Manager can move the Cost factors modeled KPI.

## KPI summary

- **Cost factors modeled**: 5-8 visible costs → 25+ including hidden
- **TCO analysis time**: 1-2 weeks → 4 hours
- **Lowest-price selection rate**: 70% default to cheapest → Decision based on true TCO
