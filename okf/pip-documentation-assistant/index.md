# PIP Documentation Assistant

> A-603 • Employee Relations

## Overview

- **Persona:** Manager
- **Department:** hr
- **Objective:** Legally-reviewed PIP template generation with SMART milestones and clear expectations. Automated progress tracking with structured evidence collection at each checkpoint. so the Manager can move the PIP drafting KPI.

## KPI summary

- **PIP drafting**: 3 hours → 15 min
- **Legal review needs**: Every PIP → Flagged only
- **Milestone tracking**: Manual → Automated
