# Product Profitability Analyzer

> A-2703 • Revenue & Cost Accounting

## Overview

- **Persona:** Controller
- **Department:** finance
- **Objective:** Fully-loaded product P&L assembled automatically from SAP CO and SD data on close day. Gemini decomposes margin changes into price, volume, mix, and cost components with root causes. so the Controller can move the Analysis turnaround KPI.

## KPI summary

- **Analysis turnaround**: 5-7 days post-close → Same-day
- **Margin root cause depth**: Top-level only → Price/volume/mix/cost decomposition
- **Pricing action speed**: Quarterly review → Monthly with recommendations
