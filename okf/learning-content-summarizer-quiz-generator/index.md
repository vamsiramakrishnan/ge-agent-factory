# Learning Content Summarizer & Quiz Generator

> A-503 • L&D

## Overview

- **Persona:** Instructional Designer
- **Department:** hr
- **Objective:** Auto-summarizes long-form learning content into bite-sized microlearning modules. Generates competency-mapped quiz questions with difficulty calibration. so the Instructional Designer can move the Summarization time KPI.

## KPI summary

- **Summarization time**: 4 hours → 5 min
- **Quiz generation**: Manual → Auto
- **Format repurposing**: 1 format → 5+ formats
