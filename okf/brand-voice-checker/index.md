# Brand Voice Checker

> A-3007 \u2022 Content & Creative

## Overview

- **Persona:** Brand Manager
- **Department:** marketing
- **Objective:** Gemini assesses whether content sounds like the brand, catching tonal mismatches that go beyond prohibited terms. Understands the difference between appropriately formal (whitepaper) and inappropriately stiff (blog post) within the same brand voice. so the Brand Manager can move the Brand compliance rate KPI.

## KPI summary

- **Brand compliance rate**: 72% → 94%
- **Review turnaround**: 1-2 days → < 5 minutes
- **Revision cycles for voice**: 2-3 rounds → 0-1 rounds
