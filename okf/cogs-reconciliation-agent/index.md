# COGS Reconciliation Agent

> A-2707 • Revenue & Cost Accounting

## Overview

- **Persona:** Cost Accountant
- **Department:** finance
- **Objective:** Automated extraction and alignment of CO and FI COGS data eliminates manual data gathering. Gemini traces breaks to source documents and generates correcting entries automatically. so the Cost Accountant can move the Reconciliation time KPI.

## KPI summary

- **Reconciliation time**: 2-3 days → < 4 hours
- **Unresolved breaks**: 5-10% of COGS → < 0.5%
- **Correcting entry accuracy**: Manual, error-prone → Auto-generated with audit trail
