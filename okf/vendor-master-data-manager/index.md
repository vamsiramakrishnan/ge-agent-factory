# Vendor Master Data Manager

> A-2205 • Accounts Payable

## Overview

- **Persona:** AP Manager
- **Department:** finance
- **Objective:** Automated TIN verification, D&B validation, and bank detail screening at onboarding. ML-based duplicate detection catches name variants and related entities before they enter the master. so the AP Manager can move the Vendor onboarding time KPI.

## KPI summary

- **Vendor onboarding time**: 5-7 days → 1 day
- **Duplicate vendors**: 8% of master → < 1%
- **Fraudulent vendor risk**: Discovered post-payment → Blocked at onboarding
