# Services Procurement & SOW Manager

> A-1806 • Indirect & Tail Spend

## Overview

- **Persona:** Procurement Lead
- **Department:** procurement
- **Objective:** Rate card compliance engine validates every timesheet against contracted rates, roles, and budget caps in real time. LLM detects scope creep by comparing timesheet narratives against SOW scope: 'SOW covers Phase 1 system design but timesheets show user acceptance testing support — this is Phase 3 work charged against Phase 1.' so the Procurement Lead can move the Scope creep detection KPI.

## KPI summary

- **Scope creep detection**: Caught at renewal → Flagged within 1 week
- **Rate card compliance**: 68% audited annually → 100% validated per timesheet
- **SOW budget overrun**: 22% avg overrun → <5% with early warning
