# Access Provisioning Orchestrator

> A-4602 • End User Computing

## Overview

- **Persona:** End User Support Lead
- **Department:** it
- **Objective:** Ingest HR events from Workday, match role-based access templates from Okta, enrich with BigQuery peer patterns, use Gemini to resolve edge cases, provision across Okta, Google Workspace, and ServiceNow, create audit tickets, and route elevated access to manager approval.

## KPI summary

- **Day-one readiness**: 60% have all access → 95% fully provisioned
- **Provisioning time**: 2-3 days → < 4 hours
- **Access-related tickets**: 35 per week → 5 per week
