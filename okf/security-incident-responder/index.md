# Security Incident Responder

> IT4-08 • Cybersecurity & Threat Management

## Overview

- **Persona:** CISO / Security Analyst
- **Department:** it
- **Objective:** Gemini reconstructs the attack kill chain in minutes using MITRE ATT&CK mapping across all security telemetry. LLM generates tailored response playbooks based on specific attack type, affected assets, and business criticality. so the CISO / Security Analyst can move the Mean time to containment KPI.

## KPI summary

- **Mean time to containment**: 4-6 hours → <45 minutes
- **Kill chain reconstruction**: Days of forensics → Minutes automated
- **Playbook coverage**: 12 static runbooks → Dynamic per-incident
