# Commodity Price Forecaster

> A-1903 • Spend Analytics

## Overview

- **Persona:** Category Manager
- **Department:** procurement
- **Objective:** Time-series forecasting across 40+ commodity indices with volatility modeling and hedging window identification. Gemini interprets market-moving events that quantitative models cannot capture and connects them to procurement impact. so the Category Manager can move the Forecast accuracy (30-day) KPI.

## KPI summary

- **Forecast accuracy (30-day)**: Analyst gut feel → 85%+ directional accuracy
- **Indices tracked**: 5-10 manual → 40+ automated
- **Time to procurement action**: Days after event → Same-day alert
