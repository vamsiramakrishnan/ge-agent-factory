# Fraud Detection Engine

> A-2606 • Internal Audit & Controls

## Overview

- **Persona:** Chief Audit Executive
- **Department:** finance
- **Objective:** Continuous monitoring with Benford's Law and behavioral analytics detects anomalies in real-time. Network analysis uncovers hidden relationships — shared bank accounts, vendor-employee connections. so the Chief Audit Executive can move the Detection frequency KPI.

## KPI summary

- **Detection frequency**: Annual review → Continuous monitoring
- **Detection-to-investigation**: Weeks-months → < 24 hours
- **Fraud schemes detected**: 3-4 types → 15+ pattern types
