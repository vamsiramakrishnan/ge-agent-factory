# Intelligent Ticket Router

> IT5-01 • IT Service Management

## Overview

- **Persona:** IT Service Desk Manager
- **Department:** it
- **Objective:** Gemini understands urgency signals in natural language — 'board presentation in 30 minutes' triggers VIP protocol. LLM classifies tickets with 92% first-route accuracy, reducing rerouting and resolution time. so the IT Service Desk Manager can move the Routing accuracy KPI.

## KPI summary

- **Routing accuracy**: 65% first-route correct → 92% first-route correct
- **Ticket triage time**: 8-12 min manual → Instant automated
- **Misroute rate**: 35% rerouted → <8% rerouted
