# Attrition Analytics Agent

> A-809 • Offboarding

## Overview

- **Persona:** CHRO
- **Department:** hr
- **Objective:** Score employee and team flight risk using BigQuery ML attrition models, identify root-cause drivers via engagement and tenure signals, and recommend data-driven retention interventions with ROI tracking — never make termination decisions or expose individual PII to non-HR users.

## KPI summary

- **Prediction accuracy**: None → 85% at 90 days
- **Root cause analysis**: Exit interviews → Multi-variable modeling
- **Intervention ROI**: Unknown → Tracked per action
