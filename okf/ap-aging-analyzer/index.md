# AP Aging Analyzer

> A-2204 • Accounts Payable

## Overview

- **Persona:** AP Manager
- **Department:** finance
- **Objective:** Auto-refreshed AP aging with DPO trending and vendor concentration analysis. Disputed invoices tracked with resolution timelines and working capital impact assessment. so the AP Manager can move the Aging report time KPI.

## KPI summary

- **Aging report time**: Half-day manual → Auto-refreshed
- **Dispute visibility**: Spreadsheet tracking → Real-time with context
- **Cash forecast accuracy**: Rough estimate → Data-driven projection
