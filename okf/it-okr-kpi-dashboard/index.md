# IT OKR & KPI Dashboard

> A-3806 • IT Strategy & Portfolio

## Overview

- **Persona:** CIO / CTO
- **Department:** it
- **Objective:** Gemini generates weekly performance narratives that explain why KPIs moved, not just that they moved. Cross-metric correlation reveals leading indicators — deployment frequency predicting MTTR improvements. so the CIO / CTO can move the Dashboard prep time KPI.

## KPI summary

- **Dashboard prep time**: 4 hours/week → Automated
- **KPI sources integrated**: 3 manual → 8+ automated
- **Root cause commentary**: None → AI-generated weekly
