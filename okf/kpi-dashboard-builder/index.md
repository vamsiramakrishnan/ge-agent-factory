# KPI Dashboard Builder

> A-2803 • Finance Analytics & Reporting

## Overview

- **Persona:** Financial Reporting Manager
- **Department:** finance
- **Objective:** Real-time dashboard refresh with automated anomaly detection on 50+ financial KPIs. Gemini generates narrative commentary explaining the 'why' behind metric movements. so the Financial Reporting Manager can move the Dashboard refresh KPI.

## KPI summary

- **Dashboard refresh**: Weekly manual → Real-time automated
- **KPI commentary**: None — charts only → AI-generated narratives
- **Anomaly detection**: Manual spot-check → Automated threshold monitoring
