# Capability Assessment Agent

> A-1306 • Supplier Discovery

## Overview

- **Persona:** Sourcing Specialist
- **Department:** procurement
- **Objective:** Weighted scoring against 40+ technical and commercial criteria applied consistently across all respondents. LLM evaluates narrative claims in context — supplier claims 'state-of-the-art quality' but PPM data shows 450 vs. incumbent's 120. so the Sourcing Specialist can move the Assessment turnaround KPI.

## KPI summary

- **Assessment turnaround**: 1-2 weeks → < 1 day
- **Criteria evaluated per supplier**: 10-15 manual → 40+ automated
- **Claim-vs-data discrepancy detection**: Rare → Systematic
