# SIEM Alert Triage Agent

> IT4-03 • Cybersecurity & Threat Management

## Overview

- **Persona:** Security Analyst
- **Department:** it
- **Objective:** Gemini auto-classifies 92% of alerts as benign with evidence, surfacing only true threats for human review. LLM reasons about alert context — distinguishing VPN artifacts from credential compromise in seconds. so the Security Analyst can move the Alert triage time KPI.

## KPI summary

- **Alert triage time**: 15-30 min each → <1 min automated
- **False positive rate**: 90% manual review → 92% auto-classified
- **Analyst alert fatigue**: 200+ daily alerts → 15 curated for review
