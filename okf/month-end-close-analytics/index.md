# Month-End Close Analytics

> A-2107 • GL & Close

## Overview

- **Persona:** Controller
- **Department:** finance
- **Objective:** Aggregate close metrics from BlackLine and BigQuery, identify bottleneck tasks and owners, compute trend vs target days, generate retrospective narrative with root-cause analysis and recommendations via Gemini, and publish refreshed Looker dashboards and distributed retrospective reports.

## KPI summary

- **Close cycle improvement**: No tracking → Measured monthly
- **Bottleneck visibility**: Anecdotal → Data-driven
- **Repeat delays**: Same issues recur → Addressed proactively
