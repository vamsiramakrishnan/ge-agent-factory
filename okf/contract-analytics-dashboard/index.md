# Contract Analytics Dashboard

> A-1408 • Contract Lifecycle

## Overview

- **Persona:** Contract Manager
- **Department:** procurement
- **Objective:** Gemini generates narrative commentary: 'Cycle time increased 40% driven by 3 board-approval deals, not systemic slowdown — excluding outliers, improved 5%.' LLM identifies negotiation patterns: 'Suppliers consistently push back on IP ownership in professional services — pre-negotiate standard IP framework.' so the Contract Manager can move the Reporting cycle KPI.

## KPI summary

- **Reporting cycle**: Monthly manual → Weekly automated
- **Insight depth**: Raw metrics only → Narrative + root cause
- **Negotiation pattern visibility**: None → Clause-level trends
