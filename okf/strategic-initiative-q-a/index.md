# Strategic Initiative Q&A

> A-3807 • IT Strategy & Portfolio

## Overview

- **Persona:** CIO / CTO
- **Department:** it
- **Objective:** Gemini provides instant, data-backed answers by querying Confluence, Jira, and BigQuery simultaneously. Every answer includes real-time status with citations — not last month's status report. so the CIO / CTO can move the Time to answer KPI.

## KPI summary

- **Time to answer**: Hours (manual research) → <30 seconds
- **Data sources queried**: 1-2 manual → 5+ automated
- **Answer accuracy**: Variable → 95%+ with citations
