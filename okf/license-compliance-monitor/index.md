# License Compliance Monitor

> A-4504 • IT Governance & Compliance

## Overview

- **Persona:** IT Procurement / Vendor Mgr
- **Department:** it
- **Objective:** Gemini merges license entitlements with actual usage data from SAM, Zylo, and Okta into a unified compliance view. LLM reasons about optimization strategies — ULA conversion vs. workload migration vs. license reclamation. so the IT Procurement / Vendor Mgr can move the Audit exposure KPI.

## KPI summary

- **Audit exposure**: $500K+ true-up risk → < $50K managed variance
- **Unused license reclaim**: Manual quarterly review → Continuous auto-detection
- **License cost optimization**: Reactive to renewals → $300K+ annual savings
