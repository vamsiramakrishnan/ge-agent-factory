# Management Reporting Agent

> A-2802 • Finance Analytics & Reporting

## Overview

- **Persona:** Financial Reporting Manager
- **Department:** finance
- **Objective:** Gemini generates audience-tailored reports — strategic for CEO, operational for COO, financial for CFO. Automated KPI aggregation delivers reports on close day instead of days later. so the Financial Reporting Manager can move the Report preparation KPI.

## KPI summary

- **Report preparation**: 2-3 days post-close → Same-day automated
- **Audience variants**: 1 generic report → 3 tailored versions
- **Commentary drafting**: Manual by FP&A → AI-generated with review
