# Liquidity Dashboard

> A-2407 - Treasury & Cash

## Overview

- **Persona:** Treasurer
- **Department:** finance
- **Objective:** Real-time cash position aggregation replaces morning manual assembly, providing intraday visibility across 45+ accounts. Gemini generates actionable treasury briefing: not just 'here are the balances' but 'sweep $15M EUR to fund next week's payment.' so the Treasurer can move the Cash visibility KPI.

## KPI summary

- **Cash visibility**: End-of-day, next morning → Real-time / intraday
- **Briefing preparation**: 45 min manual → Auto-generated daily
- **Idle cash identified**: Quarterly review → Daily sweep recommendations
