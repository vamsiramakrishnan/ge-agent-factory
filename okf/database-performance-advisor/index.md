# Database Performance Advisor

> A-4006 • Infra & Cloud Ops

## Overview

- **Persona:** SRE Manager
- **Department:** it
- **Objective:** Gemini suggests optimizations with cross-service pattern matching — applying successful fixes from one service to similar patterns elsewhere. Weekly proactive scanning catches missing indexes and query degradation before users experience slowness. so the SRE Manager can move the Slow query resolution KPI.

## KPI summary

- **Slow query resolution**: Days of investigation → Minutes with fix
- **Missing index detection**: Post-incident → Proactive weekly
- **Query performance improvement**: Ad hoc → Systematic 30x gains
