# Headcount Planning Agent

> A-2006 • FP&A

## Overview

- **Persona:** FP&A Director
- **Department:** finance
- **Objective:** Automated HRIS-to-financial-plan sync with real-time misalignment detection. Gemini interprets hiring justifications and flags requests that exceed revenue forecast capacity. so the FP&A Director can move the Plan reconciliation time KPI.

## KPI summary

- **Plan reconciliation time**: 2-3 days → 2 hours
- **Misalignments detected**: At budget review → Proactively flagged
- **Cost accuracy**: Base salary only → Fully-loaded
