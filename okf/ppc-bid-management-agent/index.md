# PPC Bid Management Agent

> A-3203 • Digital Marketing & SEO/SEM

## Overview

- **Persona:** SEO/SEM Specialist
- **Department:** marketing
- **Objective:** Daily automated bid optimization with quality score prediction and dayparting models. Gemini interprets competitive auction dynamics and explains CPC changes in context. so the SEO/SEM Specialist can move the ROAS improvement KPI.

## KPI summary

- **ROAS improvement**: Manual bids → +28% ROAS
- **Bid adjustment frequency**: Weekly manual → Daily automated
- **Budget pacing accuracy**: ±15% variance → ±3% variance
