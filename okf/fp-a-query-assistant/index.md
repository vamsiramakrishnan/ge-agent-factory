# FP&A Query Assistant

> A-2009 • FP&A

## Overview

- **Persona:** FP&A Analyst / Business Partner
- **Department:** finance
- **Objective:** Gemini interprets ambiguous questions and auto-determines the right metrics, periods, and granularity. NL-to-SQL translation provides instant answers from the financial data warehouse. so the FP&A Analyst / Business Partner can move the Query response time KPI.

## KPI summary

- **Query response time**: Hours to days → Seconds
- **Self-service rate**: 10% (analyst bottleneck) → 80%+
- **Answer quality**: Raw data tables → Contextual narrative
