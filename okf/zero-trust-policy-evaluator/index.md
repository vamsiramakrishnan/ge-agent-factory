# Zero Trust Policy Evaluator

> IT4-09 • Cybersecurity & Threat Management

## Overview

- **Persona:** CISO / Security Analyst
- **Department:** it
- **Objective:** Gemini continuously evaluates access policies against zero trust principles across all three layers. LLM prioritizes migration of legacy applications, focusing on PII-handling and internet-facing services. so the CISO / Security Analyst can move the Zero trust coverage KPI.

## KPI summary

- **Zero trust coverage**: Assessed annually → Continuous scoring
- **Legacy VPN exceptions**: Unknown count → Tracked with migration plan
- **Policy-to-application mapping**: Spreadsheet-based → Automated real-time
