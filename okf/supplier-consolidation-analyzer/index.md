# Supplier Consolidation Analyzer

> A-1307 • Supplier Discovery

## Overview

- **Persona:** Category Manager
- **Department:** procurement
- **Objective:** Graph analytics maps supplier-category-BU-geography relationships to surface consolidation clusters automatically. LLM reasons beyond numbers — 'Plant A's supplier has ITAR certification for defense contracts; cannot consolidate to the cheaper supplier.' so the Category Manager can move the Consolidation analysis time KPI.

## KPI summary

- **Consolidation analysis time**: 4-6 weeks → 3 days
- **Savings from consolidation**: Estimated in ranges → Modeled with confidence intervals
- **Constraint identification**: Discovered during execution → Pre-identified in analysis
