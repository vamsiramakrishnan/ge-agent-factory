# Debt Covenant Tracker

> A-2405 - Treasury & Cash

## Overview

- **Persona:** Treasurer
- **Department:** finance
- **Objective:** Continuous covenant monitoring with 2-quarter forward projections replaces quarterly surprise discoveries. Gemini reads loan agreements to automatically identify qualifying add-backs and interpret complex definitions. so the Treasurer can move the Covenant monitoring KPI.

## KPI summary

- **Covenant monitoring**: Quarterly manual → Continuous automated
- **Breach early warning**: None (reactive) → 2-quarter forward view
- **Add-back identification**: Manual review of charges → Automated with agreement reference
