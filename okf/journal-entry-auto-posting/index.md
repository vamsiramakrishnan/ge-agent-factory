# Journal Entry Auto-Posting

> A-2101 • GL & Close

## Overview

- **Persona:** GL Accountant / Controller
- **Department:** finance
- **Objective:** Pattern matching auto-posts 92% of journal entries without manual intervention. Gemini reads approval context to correctly classify non-standard transactions automatically. so the GL Accountant / Controller can move the Auto-post rate KPI.

## KPI summary

- **Auto-post rate**: 40% manual → 92% automated
- **Exception resolution**: Hours per item → Minutes
- **Posting errors**: 3-5% error rate → <0.5%
