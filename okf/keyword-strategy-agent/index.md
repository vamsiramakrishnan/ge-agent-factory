# Keyword Strategy Agent

> A-3202 • Digital Marketing & SEO/SEM

## Overview

- **Persona:** SEO/SEM Specialist
- **Department:** marketing
- **Objective:** Gemini analyzes search intent behind keyword clusters — distinguishing informational, commercial, and transactional queries. Continuous trend monitoring surfaces emerging opportunities before competitors. so the SEO/SEM Specialist can move the Keywords tracked KPI.

## KPI summary

- **Keywords tracked**: 500 manual → 25K+ automated
- **Gap identification**: Quarterly → Continuous
- **Content pipeline coverage**: 40% → 90%
