# Redline Comparison Agent

> A-1405 • Contract Lifecycle

## Overview

- **Persona:** Contract Manager
- **Department:** procurement
- **Objective:** Gemini performs semantic diff — understanding that 'best efforts' to 'commercially reasonable efforts' is a meaningful commitment reduction. LLM detects section deletions that remove protections and flags as likely intentional risk reallocation, not editing error. so the Contract Manager can move the Review time per redline KPI.

## KPI summary

- **Review time per redline**: 2-4 hours → 20 minutes
- **Semantic changes caught**: Word-level only → Meaning-level
- **Deleted protections missed**: 15-20% → <1%
