# Standard Cost Variance Agent

> A-2704 • Revenue & Cost Accounting

## Overview

- **Persona:** Cost Accountant
- **Department:** finance
- **Objective:** Gemini investigates significant variances by cross-referencing commodity markets and supplier benchmarks. Automated decomposition into price/usage/efficiency/volume components on close day. so the Cost Accountant can move the Variance analysis time KPI.

## KPI summary

- **Variance analysis time**: 3-4 days → Same-day
- **Root cause identification**: Manual investigation → Automated with market correlation
- **Standard cost accuracy**: Annual update → Quarterly with variance-driven refresh
