# Board Deck Generator

> A-2008 • FP&A

## Overview

- **Persona:** CFO
- **Department:** finance
- **Objective:** Automated KPI aggregation and chart assembly from 15+ data sources into slide templates. Gemini generates management discussion narratives tailored to each audience — risk vs. strategy focus. so the CFO can move the Deck preparation time KPI.

## KPI summary

- **Deck preparation time**: 3-5 days → 4 hours
- **Data sources integrated**: 5-6 manual → 15+ automated
- **Narrative quality**: Copy-paste charts → Audience-tailored
