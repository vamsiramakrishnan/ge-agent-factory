# OKR Progress Tracker Agent

> A-302 • Performance Mgmt

## Overview

- **Persona:** Manager
- **Department:** hr
- **Objective:** Continuous OKR progress monitoring from integrated data sources. Automated nudges for stalled goals and at-risk key results. so the Manager can move the Update frequency KPI.

## KPI summary

- **Update frequency**: Quarterly → Continuous
- **Stalled goal detection**: At review → Real-time
- **Recalibration speed**: Next quarter → Immediate
