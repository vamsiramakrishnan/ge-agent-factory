# IaC Drift Detector

> A-3909 • Software Engineering & DevOps

## Overview

- **Persona:** DevOps Lead
- **Department:** it
- **Objective:** Gemini explains drift with incident context — distinguishing emergency workarounds that should be imported from errors to remove. Daily drift detection catches manual changes within 24 hours instead of letting them accumulate for months. so the DevOps Lead can move the Drift detection frequency KPI.

## KPI summary

- **Drift detection frequency**: Quarterly audit → Daily automated
- **Manual changes in production**: Unknown → 100% tracked
- **State reconciliation time**: Days per environment → Auto-generated PRs
