# Sourcing Channel Optimizer

> A-1212 • Strategic Sourcing

## Overview

- **Persona:** Indirect Procurement Lead
- **Department:** procurement
- **Objective:** Gemini reasons about edge cases: '$8K recurring custom packaging — custom spec makes it supplier-dependent, not catalog-eligible despite recurrence.' LLM analyzes the trade-off between channel efficiency and category leverage, preventing inappropriate channel migration. so the Indirect Procurement Lead can move the Catalog adoption rate KPI.

## KPI summary

- **Catalog adoption rate**: 35% → 72%
- **Buyer hours saved/year**: Baseline → 1,200+ hours
- **Maverick transactions migrated**: Ad hoc → 340+ per quarter
