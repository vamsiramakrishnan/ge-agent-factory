# Ad-Hoc Query Agent

> A-2804 • Finance Analytics & Reporting

## Overview

- **Persona:** FP&A Analyst
- **Department:** finance
- **Objective:** Natural language interface makes financial data accessible to any business user. Gemini interprets ambiguous questions and returns answers with business context and citations. so the FP&A Analyst can move the Query response time KPI.

## KPI summary

- **Query response time**: Hours-days (manual pull) → < 30 seconds
- **Data accessibility**: Requires SQL knowledge → Natural language
- **Context quality**: Raw numbers only → Numbers + explanations + next steps
