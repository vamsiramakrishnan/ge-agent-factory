# New Hire Q&A Assistant

> A-212 • Onboarding & Integration

## Overview

- **Persona:** New Hire
- **Department:** hr
- **Objective:** AI-powered conversational assistant providing instant, accurate answers to new hire questions 24/7. Contextual responses drawn from the full policy corpus, tailored to the employee's role, location, and start date. so the New Hire can move the Response time KPI.

## KPI summary

- **Response time**: 24-48 hrs → Instant
- **HR ticket volume**: 100% → -60%
- **Answer accuracy**: Varies → 95%+
