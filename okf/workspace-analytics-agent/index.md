# Workspace Analytics Agent

> A-4603 • End User Computing

## Overview

- **Persona:** End User Support Lead
- **Department:** it
- **Objective:** Gemini aggregates usage metrics across Google Workspace, Microsoft 365, and Slack into a unified view. LLM synthesizes quantitative data into actionable insights — identifying tool overlap and shadow IT signals. so the End User Support Lead can move the License waste identified KPI.

## KPI summary

- **License waste identified**: Unknown → $180K/year reclaimed
- **Tool overlap detection**: Ad-hoc complaints → Automated monthly
- **Shadow IT visibility**: Discovered post-incident → Proactive detection
