# Developer Experience Surveyor

> A-3908 • Software Engineering & DevOps

## Overview

- **Persona:** VP Engineering
- **Department:** it
- **Objective:** Gemini connects DORA metrics with developer sentiment — explaining why deployment frequency improved but satisfaction dropped. Slack sentiment analysis surfaces tooling friction in real-time rather than waiting for annual surveys. so the VP Engineering can move the DORA metrics visibility KPI.

## KPI summary

- **DORA metrics visibility**: None → Continuous tracking
- **Friction point detection**: Annual survey → Monthly automated
- **DevEx improvement cycle**: Reactive → Data-driven monthly
