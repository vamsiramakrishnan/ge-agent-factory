# Goal Drafting & Alignment Assistant

> A-301 • Performance Mgmt

## Overview

- **Persona:** Manager
- **Department:** hr
- **Objective:** Generates SMART goal drafts from role context and job descriptions. Auto-cascades objectives from company OKRs through every org level. so the Manager can move the Goal quality KPI.

## KPI summary

- **Goal quality**: Vague → SMART-verified
- **Alignment score**: 40% → 95%
- **Drafting time**: 2 hours → 5 min
