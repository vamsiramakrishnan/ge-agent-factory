# Savings Pipeline Tracker

> A-1105 • Procurement Strategy

## Overview

- **Persona:** CPO
- **Department:** procurement
- **Objective:** ML classifies savings types (hard negotiated, cost avoidance, demand reduction) with realization probability scoring. LLM interprets why savings leaked — reads PO exception context to distinguish capacity issues from compliance failures. so the CPO can move the Savings realization rate KPI.

## KPI summary

- **Savings realization rate**: 40-50% → 85%+
- **Leakage detection**: Quarterly manual → Weekly automated
- **Pipeline visibility**: Spreadsheet → Real-time dashboard
