# AR Aging & DSO Analyzer

> A-2305 - AR & Collections

## Overview

- **Persona:** AR Manager
- **Department:** finance
- **Objective:** Automated segment-level DSO analysis reveals that a 5-day DSO increase is driven entirely by 2 government contracts, not a systemic issue. Gemini generates narrative context that transforms numbers into actionable intelligence for the AR Manager. so the AR Manager can move the Reporting cycle KPI.

## KPI summary

- **Reporting cycle**: Manual weekly → Auto-refreshed daily
- **Root cause visibility**: Top-line DSO only → Segment-level with narrative
- **Bad debt forecast accuracy**: 60% → 88%
