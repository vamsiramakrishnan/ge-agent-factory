# Risk Assessment Agent

> A-2604 • Internal Audit & Controls

## Overview

- **Persona:** Chief Audit Executive
- **Department:** finance
- **Objective:** Gemini synthesizes quantitative risk scores with qualitative signals from multiple data sources. Correlation analysis identifies risk clusters that individual assessments miss. so the Chief Audit Executive can move the Risk assessment cycle KPI.

## KPI summary

- **Risk assessment cycle**: 8-12 weeks → 2-3 weeks
- **Risk signals analyzed**: 5-10 manual sources → 50+ automated sources
- **Audit plan coverage**: Risk-judgmental → Data-driven prioritization
