# IT Budget Forecast Agent

> A-3803 • IT Strategy & Portfolio

## Overview

- **Persona:** CIO / CTO
- **Department:** it
- **Objective:** Gemini explains variances with business context — separating one-time events from structural cost trends. Seasonal decomposition on cloud spend prevents overreaction to expected periodic spikes. so the CIO / CTO can move the Forecast accuracy KPI.

## KPI summary

- **Forecast accuracy**: 70-75% → 93%
- **Budget prep time**: 2 weeks → 1 day
- **Variance explanation**: Manual research → Auto-generated
