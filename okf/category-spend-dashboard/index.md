# Category Spend Dashboard

> A-1208 • Strategic Sourcing

## Overview

- **Persona:** Category Manager
- **Department:** procurement
- **Objective:** Gemini generates plain-English commentary per dashboard section: 'IT hardware up 35% QoQ — driven by data center refresh, not organic growth. Normalizes in Q3.' LLM cross-references spend anomalies with project approvals, seasonal patterns, and contract changes to explain the why, not just the what. so the Category Manager can move the Dashboard refresh lag KPI.

## KPI summary

- **Dashboard refresh lag**: Manual monthly → Automated weekly
- **Anomalies explained**: 0% auto-explained → 85% with root cause
- **Time to insight**: Half-day of SQL → Open and read
