# Supplier Diversity Tracker

> A-1304 • Supplier Discovery

## Overview

- **Persona:** Supplier Development Mgr
- **Department:** procurement
- **Objective:** Spend attribution engine maps every dollar to certified diverse suppliers with MBE/WBE/SDVOB/HUBZone classification. Tier-2 reporting aggregated from prime supplier submissions with certification cross-validation. so the Supplier Development Mgr can move the Diversity report generation KPI.

## KPI summary

- **Diversity report generation**: 2 weeks manual → Automated monthly
- **Certification accuracy**: ~80% (stale data) → 99%+ real-time sync
- **Tier-2 spend visibility**: None → Full sub-tier tracking
