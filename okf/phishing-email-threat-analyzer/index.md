# Phishing & Email Threat Analyzer

> IT4-04 • Cybersecurity & Threat Management

## Overview

- **Persona:** Security Analyst
- **Department:** it
- **Objective:** Gemini detects BEC impersonation by analyzing writing style against known executive communication patterns. LLM identifies social engineering tactics and domain spoofing that bypass traditional filters. so the Security Analyst can move the Phishing analysis time KPI.

## KPI summary

- **Phishing analysis time**: 25 min per email → 3 min automated
- **BEC detection rate**: 60% caught → 95% with LLM style analysis
- **Org-wide quarantine speed**: 2-4 hours → <5 minutes
