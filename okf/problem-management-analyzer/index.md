# Problem Management Analyzer

> IT5-06 • IT Service Management

## Overview

- **Persona:** IT Service Desk Manager
- **Department:** it
- **Objective:** Gemini clusters incidents automatically and identifies systemic root causes across services weekly. LLM generates actionable problem statements connecting repeat incidents to shared infrastructure bottlenecks. so the IT Service Desk Manager can move the Repeat incident rate KPI.

## KPI summary

- **Repeat incident rate**: 25% recurrence → 8% with root cause fixes
- **Problem identification**: Manual quarterly review → Automated weekly
- **MTTR for known issues**: 45 min avg → 15 min with KB match
