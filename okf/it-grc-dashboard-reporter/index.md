# IT GRC Dashboard & Reporter

> A-4507 • IT Governance & Compliance

## Overview

- **Persona:** CIO / CTO
- **Department:** it
- **Objective:** Gemini aggregates GRC metrics from ServiceNow, RSA Archer, and OneTrust into a unified cross-domain view. LLM generates executive narratives that translate technical GRC status into board-ready business language. so the CIO / CTO can move the GRC report preparation KPI.

## KPI summary

- **GRC report preparation**: 5 days manual compilation → Auto-generated monthly
- **Metric coverage**: 60% of GRC domains → 100% cross-domain
- **Board report turnaround**: 2 weeks → Same day
