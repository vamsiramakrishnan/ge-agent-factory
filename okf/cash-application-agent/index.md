# Cash Application Agent

> A-2301 - AR & Collections

## Overview

- **Persona:** AR Specialist
- **Department:** finance
- **Objective:** ML model trained on 3 years of application history achieves 95%+ auto-match rate, freeing staff for exception management. Gemini reads remittance notes and cross-references customer agreements to validate deductions without customer calls. so the AR Specialist can move the Auto-match rate KPI.

## KPI summary

- **Auto-match rate**: 55-65% → 95%+
- **Deduction resolution time**: 5-7 days → Same day
- **Cash application backlog**: 3-4 days → < 4 hours
