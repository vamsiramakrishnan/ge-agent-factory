# Data Quality Scorecard

> IT6-02 • Data & AI Platform

## Overview

- **Persona:** Data Platform Lead
- **Department:** it
- **Objective:** Gemini explains quality degradations with root cause hypotheses tied to upstream system changes. LLM correlates quality drops with recent data pipeline or application changes automatically. so the Data Platform Lead can move the Quality coverage KPI.

## KPI summary

- **Quality coverage**: 40% of tables checked → 95% of tables checked
- **Degradation detection**: Reported by consumers → Proactive daily alerts
- **Root cause identification**: Hours of investigation → Auto-hypothesized
