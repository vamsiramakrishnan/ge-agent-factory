# Peer Benchmarking Agent

> A-2805 • Finance Analytics & Reporting

## Overview

- **Persona:** CFO
- **Department:** finance
- **Objective:** Automated peer data collection and ratio calculation across 30+ metrics with percentile ranking. Gemini contextualizes gaps — distinguishes strategic choices from genuine underperformance. so the CFO can move the Benchmarking cycle KPI.

## KPI summary

- **Benchmarking cycle**: 2-3 weeks manual → 2-3 days automated
- **Peer metrics compared**: 5-10 basic ratios → 30+ with percentile ranking
- **Strategic context**: Numbers without explanation → AI-contextualized gap analysis
