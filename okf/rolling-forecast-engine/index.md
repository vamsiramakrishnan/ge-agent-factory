# Rolling Forecast Engine

> A-2002 • FP&A

## Overview

- **Persona:** FP&A Director
- **Department:** finance
- **Objective:** Time-series models auto-refresh monthly with statistical confidence intervals on every line. Gemini interprets earnings calls, pipeline notes, and macro shifts to adjust assumptions systematically. so the FP&A Director can move the Forecast accuracy KPI.

## KPI summary

- **Forecast accuracy**: 70-75% → 92%+
- **Forecast refresh cycle**: 2-3 weeks → 2 days
- **Qualitative signals incorporated**: Ad-hoc → Systematic
