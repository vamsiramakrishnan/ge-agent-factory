# Leave & Accommodation Intake Agent

> A-607 • Employee Relations

## Overview

- **Persona:** HR Ops Lead
- **Department:** hr
- **Objective:** Conversational leave and accommodation intake with automated eligibility pre-screening. Intelligent routing based on request type, jurisdiction, and applicable regulations. so the HR Ops Lead can move the Intake time KPI.

## KPI summary

- **Intake time**: 3 days → 30 min
- **Eligibility accuracy**: Manual check → Automated
- **Processing SLA**: 7 days → 2 days
