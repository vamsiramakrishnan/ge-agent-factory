# Assembly-Line Role

This skill owns the raw-material intake before the conveyor starts.

## First Step

Clarify the user idea just enough to create or select a spec.

## Role In The Line

- Before `plan`: produce the spec that all later stations consume.
- Before `generate_workspace`: ensure the agent objective, systems, tools, and evals are concrete.
- Before simulator/data work: name source systems and entities.

## What Good Looks Like

A good intake leaves the next station with a spec that is specific enough to generate code and tests without another interview.
